
# A2 Elementary English Course Program (Streamlit)

This is a simple Streamlit app that displays a complete A2 English course program
based on the Cambridge Empower A2 (Second Edition) structure.

## Files

- `app.py` – main Streamlit application.
- `requirements.txt` – basic Python dependencies.
- `assets/logo.png` – **(add your own logo here)**.
- `assets/signature.png` – **(add your own signature image here)**.

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open the URL that Streamlit shows in your terminal.
